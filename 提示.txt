# 角色
你是一位資深 Python 工程師與 App 應用設計師。

# 任務目標
根據以下需求，生成一個可運行的 Kivy 庫存管理 App 程式碼（.py 檔案）。

# 需求範圍（我負責的部分 - 第一頁）
庫存小分頁面：包含 [年齡]、[性別]、[身高]、[體重]、 [疾病]。

# UI 設計
1. 風格：簡約典雅
2. 物品圖示：使用 Kivy 內建 icon 或本地圖片資料夾（Flaticon 素材請自行下載放入 assets/）
3. 每個區域顯示：物品數量 + 剩餘保存期限（如：D-3）
4. 右下角：+ 按鈕（新增家人資料）
5. 最下方：工具列切換頁面

# 功能需求
1. 年齡（age）
   -性別（sex）
   - 身高（Height）
   - 體重（weight）

2. 點擊物品：查看/編輯詳細資訊
3. AI 聊天：使用 Gemini API，根據用戶資料 + 庫存生成推薦購買清單

# 資料庫（SQLite）
- 檔案名稱：inventory.db
- 資料表：inventory
  - id（主鍵）
  - name（名稱）
  - quantity（數量）
  - expiry_date（效期）
  - buy_date（購買日期）
  - area（區域）

# 新增收庫存邏輯
- 新增同名物品時：數量累加，保留最早的效期和購買日期
- 數量歸零時：該筆資料刪除

# API 設定
- 使用 Google Gemini API
- API Key：請用環境變數 GEMINI_API_KEY

# 交付物
1. main.py（完整可運行程式）
2. README.md（說明如何運行）
3. 資料夾結構說明
